Privacy Policy

Client-Side Processing

All file processing in nrepo occurs entirely in your browser. Your files are never uploaded to any server or stored anywhere outside your device.

Data Collection

We do not collect, store, or transmit any of your personal data or code. Everything stays on your local machine.

GitHub Pages Hosting

This website is hosted on GitHub Pages and adheres to their Terms of Service. While GitHub provides the hosting infrastructure, they do not have access to any of your processed files or code.

Third-Party Services

The only third-party services we use are:

  • GitHub Pages (for hosting)
  • CDN services (for loading framework files)
