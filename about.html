How to Use nrepo

What is nrepo?

nrepo is a tool designed to prepare and combine source code files for use with Large Language Models (LLMs). It helps developers easily process multiple source files into an optimized text format that LLMs can better understand and work with.

Ways to Use nrepo

1. Local Files Upload

Process files from your local machine:

  • Visit the Upload Files page
  • Drag and drop files/folders or use the "Select Files" button / There is a "Select Directory" button to upload folders
  • Choose your target LLM model and configure settings
  • Process and download the formatted output

2. Repository Processing

Convert code directly from GitHub or GitLab repositories:

  • Go to the Repositories page
  • Enter a repository URL (e.g., https://github.com/username/repo or https://gitlab.com/username/repo)
  • Select specific branches or folders to process
  • Get your code formatted for LLM use

Use Cases

  • Code Analysis: Prepare your codebase for LLM-based code review and analysis
  • Documentation: Generate comprehensive documentation using LLMs with your processed code
  • Code Understanding: Help LLMs better understand your project structure and relationships
